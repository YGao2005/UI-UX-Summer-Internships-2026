# UI/UX Design Internships 2025

> Automated job tracker for UI/UX design internships, updated daily.

This repository automatically scrapes and aggregates UI/UX design internship opportunities from multiple sources including Greenhouse, Lever, Ashby, and RemoteOK.

## Status

🚧 **Initial Setup** - Run the scraper to populate job listings!

```bash
python src/main.py
```

## Features

- Daily automated updates via GitHub Actions
- Smart filtering for UI/UX internships specifically
- Deduplication across multiple sources
- Clean, searchable table format
- Completely free (no API keys required for basic operation)

## Quick Links

- **[Setup Guide](SETUP.md)** - How to run and customize
- **[Contributing](SETUP.md#contributing)** - Add companies to track

## Data Sources

Jobs are collected from:
- Greenhouse (50+ tech companies)
- Lever (design agencies & startups)
- Ashby (Notion, OpenAI, Anthropic, Linear)
- RemoteOK (remote design internships)
