# UI/UX Internship Scraper - Project Summary

## What We Built

A fully automated, free UI/UX internship tracker that:
- Scrapes 4 job sources (Greenhouse, Lever, Ashby, RemoteOK)
- Filters for relevant UI/UX design internships
- Removes duplicates intelligently
- Updates a GitHub repo daily via GitHub Actions
- Costs $0/month to run

## Customization

### Add More Companies

Edit `data/companies.yml`:

```yaml
- name: "YourCompany"
  greenhouse: "company-handle"
```

To find handles:
- Greenhouse: Visit company.com/careers → View source → Search for "greenhouse.io"
- Lever: Check URL at company.com/careers (e.g., lever.co/v0/postings/HANDLE)
- Ashby: Check URL at company.com/careers

### Adjust Filtering

Edit `data/keywords.yml`:
- Add more title keywords
- Adjust minimum score threshold
- Add/remove excluded keywords

### Change Update Frequency

Edit `.github/workflows/update_jobs.yml`:
```yaml
schedule:
  - cron: '0 */6 * * *'  # Every 6 hours instead of daily
```

## Troubleshooting

### Common Issues

1. **"Refusing to create venv" error**
   → Directory name contains colon, rename as shown above

2. **GitHub Actions not running**
   → Check workflow permissions in Settings

3. **No jobs found**
   → Some companies may have incorrect handles, check logs

4. **Too many jobs**
   → Increase `minimum_match_score` in data/keywords.yml
